## 로또 계산기

## 기능 목록
- 로또 번호는 1~45 숫자를 가진다.
- 로또는 중복되지 않은 로또 번호 6개 + 보너스 번호로 이루어져 있다.
- 로또 당첨은 당첨 금액과 조건을 가지고 있다.
    - 번호 일치 또는 보너스 번호 일치 여부로 로또 번호를 당첨될 수 있다
        - 1등: 6개 번호 일치 / 2,000,000,000원
        - 2등: 5개 번호 + 보너스 번호 일치 / 30,000,000원
        - 3등: 5개 번호 일치 / 1,500,000원
        - 4등: 4개 번호 일치 / 50,000원
        - 5등: 3개 번호 일치 / 5,000원
- 로또 1장은 1000원이다.
- 사용자는 로또를 구매할 수 있다.
- 로또를 구매하면 금액을 차감한다.
- 당첨번호와 보너스 번호를 입력 받는다.
- 당첨 내역 및 수익률을 확인할 수 있다.
- 잘못된 입력에 대해서는 "[ERROR]"로 시작하는 에러 메세지와 함께 다시 입력 받는다
